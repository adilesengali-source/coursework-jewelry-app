
# JewelryShopApp

**JewelryShopApp** — мобильное Android-приложение для учёта и расчёта стоимости ювелирных изделий, разработанное как курсовая работа. Приложение позволяет вести каталог изделий, добавлять новые позиции, просматривать подробности, удалять позиции и рассчитывать стоимость с учётом веса и количества.

---

## Краткое описание

Проект демонстрирует типичную архитектуру небольшого Android-приложения на Kotlin: экраны для показа списка и деталей, адаптер для списка (RecyclerView), модель данных Product, навигация между экранами и утилиты для загрузки/подготовки тестовых данных. Приложение ориентировано на локальное хранение/отображение данных (в текущей реализации может использоваться заглушечный DataProvider).

---

## Основной функционал

- Просмотр каталога изделий в виде списка карточек
- Добавление нового изделия (название, цена, вес, материалы, эмодзи/иконка)
- Просмотр полной информации о выбранном изделии
- Удаление изделия из каталога
- Калькулятор стоимости изделия: изменение веса и количества и автоматический расчёт итоговой цены
- Сохранение и отображение даты добавления изделия

## Быстрый старт — установка и запуск

1. Клонируйте репозиторий:
```

git clone [https://github.com/adilesengali-source/coursework-jewelry-app.git](https://github.com/adilesengali-source/coursework-jewelry-app.git)
cd coursework-jewelry-app

```

2. Откройте проект в Android Studio (рекомендуется последняя стабильная версия).
3. Дождитесь синхронизации Gradle и установки зависимостей.
4. Запустите приложение на эмуляторе или реальном устройстве (API 21+ рекомендуется).

Сборка из командной строки:
```

./gradlew assembleDebug

```

## Описание ключевых модулей и экранов

- MainActivity.kt
  - Служит контейнером приложения. Здесь инициализируется NavController или фрагменты, настраивается тема и глобальные обработчики (например, обработка back-press). Если используется Jetpack Navigation, MainActivity содержит NavHostFragment.

- adapters/ProductAdapter.kt
  - RecyclerView.Adapter, который отвечает за привязку списка Product к визуальным карточкам. Обычно реализует ViewHolder, обрабатывает:
    - отображение названия, цены, веса, эмодзи/иконки;
    - обработку клика по карточке (переход к ProductDetailScreen);
    - опциональные действия (удаление, добавление в корзину);
    - обновление списка (notifyDataSetChanged/плюс дифф-утилиты при необходимости).

- models/Product.kt
  - Класс/дата-класс, содержащий поля продукта. Рекомендуемая структура:
    - id: String или Long
    - name: String
    - price: Double/BigDecimal
    - weight: Double
    - materials: String (или List<String>)
    - emoji/icon: String
    - dateAdded: Long или String
    - description: String (опционально)

- screens/CatalogScreen.kt
  - Экран со списком изделий. Использует ProductAdapter для отображения; содержит FAB или кнопку для перехода на AddItemScreen; поддерживает удаление и сортировку/фильтрацию (если реализовано).

- screens/ProductDetailScreen.kt
  - Отображает полную информацию по изделию: большие иконка/эмодзи, материалы, полное описание, дата добавления, кнопки Редактировать/Удалить.

- screens/AddItemScreen.kt
  - Форма для ввода данных нового изделия. Поля: название, цена, вес, материалы, опционально эмодзи/иконка. При вводе чисел автоматически пересчитывается итоговая стоимость. После сохранения объект добавляется в источник данных (DataProvider/DB) и пользователь возвращается в каталог.

- screens/CalcScreen.kt
  - Калькулятор: позволяет выбрать изделие (или принимает выбранное на вход), указать количество и вес, применить формулы/коэффициенты и получить итоговую стоимость. Полезно для расчётов индивидуальных заказов.

- utils/DataProvider.kt
  - Утилита, возвращающая стартовый список продуктов (фиктивные данные) или работающая с локальным JSON. Позволяет быстро заполнять каталог для демонстрации. В дальнейшем сюда можно подключить Room/SharedPreferences/Remote API.

- navigation/NavGraph.kt
  - Описание навигации (идентификаторы экранов и аргументы). Используется для централизованного управления переходами и передачи параметров (напр., productId).
