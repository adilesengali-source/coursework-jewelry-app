# JewelryShopApp  
Приложение для учета и расчёта стоимости ювелирных изделий

---

## Описание проекта
**JewelryShopApp** — мобильное приложение, разработанное в рамках курсовой работы.  
Оно предназначено для автоматизации учёта, хранения и расчёта стоимости ювелирных изделий в ювелирном магазине.

Приложение позволяет добавлять изделия, просматривать каталог, удалять позиции, а также рассчитывать стоимость изделия через удобный калькулятор.

---

## Основной функционал

### Каталог изделий (CatalogScreen)
- Просмотр всех изделий в виде списка карточек  
- Удаление изделий  
- Переход к подробной информации об изделии  
- Добавление нового изделия  

### Добавление изделия (AddItemScreen)
- Ввод названия, цены, веса  
- Автоматический расчёт общей стоимости  
- Сохранение даты добавления  

### Просмотр изделия (DetailScreen)
- Полная информация о выбранном изделии  
- Эмодзи-значок изделия  
- Дата добавления, материалы, стоимость  

### Калькулятор (CalcScreen)
- Выбор изделия из каталога  
- Автоматическая подстановка цены  
- Изменение веса и количества  
- Расчёт итоговой стоимости
